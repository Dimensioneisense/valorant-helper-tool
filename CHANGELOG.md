# Changelog
All notable changes to the Valorant Helper Tool will be documented in this file.

## [v1.0.0] - 2024-07-26
### Added
* Real-time game statistics display feature, providing users with up-to-date information on their gameplay
* Automated game state tracking, allowing users to focus on their gameplay without manual tracking
* Support for Windows operating system, utilizing DirectX for enhanced performance
* Customizable overlay feature, enabling users to personalize their game statistics display
* Integration with Python scripting, allowing developers to extend the tool's functionality
* User-friendly interface, providing easy access to the tool's features and settings
* Game statistics logging feature, enabling users to review their gameplay history

### Notes
* The Valorant Helper Tool is designed to work exclusively with the Windows operating system and may not be compatible with other platforms
* The tool utilizes Python scripting for extensibility, but requires a working knowledge of Python to modify or extend its functionality
* The Valorant Helper Tool is intended for use by the gaming community, and its development is focused on providing a useful utility for gamers and developers alike
