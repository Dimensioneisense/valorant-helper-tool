## How to contribute
The Valorant Helper Tool is an open-source project that relies on contributions from the gaming community to enhance its features and improve user experience. To contribute to the project, you can start by exploring the repository at https://github.com/Dimensioneisense/valorant-helper-tool and familiarizing yourself with the codebase. We encourage developers, gamers, and enthusiasts to participate in the development process, whether it's by reporting issues, suggesting new features, or contributing code.

## Reporting issues
If you encounter any issues while using the Valorant Helper Tool, please report them to help us improve the project. When reporting issues, provide as much detail as possible, including steps to reproduce the problem, error messages, and your system configuration. This information will help us identify and fix the issue quickly, ensuring that the Valorant Helper Tool continues to provide a seamless experience for gamers.

## Code style
The Valorant Helper Tool is built using Python and DirectX, and we follow standard professional guidelines for code style and formatting. When contributing code, please ensure that your changes adhere to these guidelines to maintain consistency throughout the project. This will make it easier for other contributors to understand and build upon your work.

## Pull requests
If you've made changes to the code or have new features to add, submit a pull request to the Valorant Helper Tool repository. Our team will review your changes and provide feedback to ensure that they align with the project's goals and coding standards. By contributing to the Valorant Helper Tool, you'll be helping to create a more comprehensive and user-friendly tool for the gaming community, and we appreciate your support in making this project a success.
