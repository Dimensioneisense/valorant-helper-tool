# INSTALL.md
## Prerequisites
The Valorant Helper Tool is designed to run on Windows operating systems, utilizing DirectX and Python. To ensure proper functionality, the following prerequisites must be met:
1. A 64-bit version of Windows 10 or later
2. Python 3.8 or later installed on the system
3. The latest version of DirectX

## Download
To obtain the latest version of the Valorant Helper Tool, visit the releases page on the project's GitHub repository: https://github.com/Dimensioneisense/valorant-helper-tool/releases. Download the executable file corresponding to your system architecture.

## Install
Installation of the Valorant Helper Tool is straightforward:
1. Run the downloaded executable file
2. Follow the prompts to choose the installation location
3. Wait for the installation process to complete

## First run
Upon launching the Valorant Helper Tool for the first time, you will be prompted to configure the game state tracking feature. This involves specifying the path to the Valorant game executable and selecting the desired overlay settings. Once configured, the tool will begin tracking game statistics and displaying them in real-time.

## Troubleshooting
If issues arise during installation or runtime, refer to the following steps:
1. Verify that the system meets the prerequisites outlined above
2. Check for any conflicts with other game helper tools or overlays
3. Consult the project's issue tracker on GitHub for known issues and solutions
By following these steps, you should be able to successfully install and run the Valorant Helper Tool, enhancing your gaming experience with real-time game statistics and automation features.
