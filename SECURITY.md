# SECURITY.md
The Valorant Helper Tool is committed to ensuring the security and integrity of our users' experience. We take all security vulnerabilities seriously and appreciate your help in maintaining the security of our project.

## Supported versions
The following versions of the Valorant Helper Tool are currently supported and will receive security updates:
| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Previous release | Yes, for a limited time |
| Older releases | No |

## Reporting a vulnerability
If you believe you have found a security vulnerability in the Valorant Helper Tool, please report it to us via the Issues page at https://github.com/Dimensioneisense/valorant-helper-tool/issues. When submitting a vulnerability report, please provide as much detail as possible, including steps to reproduce the issue and any relevant system or environment information.

## Disclosure policy
The Valorant Helper Tool follows a responsible disclosure policy, which means that we will work with you to verify and address the vulnerability before making any public disclosures. We will do our best to respond to your report in a timely manner and keep you informed of our progress in addressing the vulnerability.
